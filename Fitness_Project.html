Fitness App

I attended a Learn to Code workshop through the Austin Coding Academy in Austin, which was my initial introduction to a structured javascript lesson. The instructor guided the class from initializing the project in github to developing and deploying our first app. The purpose of the project was to design and build a quote generator. I was employed as a fitness professional during this time so a fitness theme seemed appropriate. Several motivational quotes were listed in the app that hold significant meaning in my life that were actively used by my clients and other people to support their fitness aspirations.

This was a very simple design using html, css, and javascript. The Get Hype title immediately grabs ones attention along with a sub-title that further speaks to the idea of the app. An image representing fitness as well as a royal blue background color draws visual appeal. The background was also selected to complement the blue color in the picture. Within the app, a green button with a “pressed” effect is positioned below the image, which generates several quotes when clicked. View the project below and GET HYPED AND MOTIVATED!!!!
